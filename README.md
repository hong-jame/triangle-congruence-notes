# 삼각형의 합동 오답노트

중학교 「삼각형의 합동의 활용」 유형을 **단일 HTML 파일**로 정리한 오답노트. 문항·그림·풀이 모두 직접 만든 자작 자료입니다.

**👉 [바로 열기](https://hong-jame.github.io/triangle-congruence-notes/)**

## 수록 문항

한 직선 위에 세 점 A, C, B 를 놓고 같은 쪽에 삼각형 두 개를 세운 뒤, AE 와 BD 의 교점 P 에서의 각을 구하는 유형. **회전 합동(△ACE ≡ △DCB)으로 각을 옮겨 푸는 것**이 핵심입니다.

| | 조건 | 구하는 것 | 답 |
|---|---|---|---|
| 본문항 | 정삼각형 ACD, CBE | ∠APB | 120° |
| 유사 1 | 같은 그림 | ∠APD | 60° |
| 유사 2 | 꼭지각 50° 이등변 두 개 | ∠APB | 130° |
| 유사 3 | ∠APB = 145° (역방향) | ∠ACD | 35° |

끼인각이 같음을 **공통각 ∠DCE 에 같은 각을 더해서** 보이는 것이 요령입니다(공통각 값은 끝까지 구할 필요가 없습니다). 결론은 `∠APB = ∠DCB = 180° − ∠ACD` — 교점에서의 각이 사실 C 에서의 각이었다는 것이고, 일반화하면 꼭지각이 θ 일 때 **∠APB = 180° − θ** 입니다.

## 기능

- 문제를 먼저 풀고 「정답 확인」으로 펼쳐 비교
- 유사문제 3개 즉시 채점 + 점수 기록
- 오답 원인 메모, 복습 완료 체크 (브라우저에 자동 저장)
- 태그·검색, 인쇄 (`🖨 문제만` / `🖨 해설까지`)

풀이는 **학생이 직접 읽는 것**을 전제로 반말·단계별 친절체로 썼습니다.

## 직접 고쳐 쓰기

`오답노트.html` 하나만 있으면 되지만, 문항을 추가하려면 소스에서 빌드합니다.

```bash
python 그림생성.py   # 그림 SVG 생성 + 답 자동 검산
python build.py      # 데이터.json + 그림/ -> 오답노트.html (그림을 base64로 박음)
```

문항은 `데이터.json`에 추가합니다. 키: `id/출처/날짜/태그[]/문제/그림/확인필요/정답/풀이[]/핵심/유사문제[]`

그림 좌표는 눈대중으로 찍지 말고 수치로 확정하는 방식입니다. `그림생성.py`는 점을 계산으로 잡아 SVG를 그린 뒤, 답(`∠APB = 180° − θ`)과 합동 조건(`CA=CD`, `CE=CB`, `∠ACE=∠DCB`, `AE=DB`)까지 자동 검산합니다.

| 파일 | 역할 |
|---|---|
| `오답노트.html` | **결과물.** 그림까지 박힌 단일 파일 (`index.html`은 Pages 루트용 사본) |
| `데이터.json` | 문항 원본 데이터 |
| `템플릿.html` | UI/스크립트. `/*__DATA__*/[]` 자리에 데이터 주입 |
| `build.py` | 데이터.json + 그림/ → 오답노트.html |
| `그림생성.py` | 그림 SVG 생성 + 답 검산 |
